# 🍉 Fruit Hunter — Stress Relief Fruit Slicing Game


## Overview

**Fruit Hunter** is a fast-paced and visually satisfying fruit slicing game designed for **stress relief, relaxation, and pure enjoyment**.
Swipe across the screen to slice flying fruits, trigger juicy visual effects, build combos, and enjoy an immersive arcade-style experience.

Unlike competitive or high-pressure games, Fruit Hunter focuses on:

* Relaxation through interaction
* Satisfying slicing animations
* Quick dopamine-reward gameplay
* Anxiety reduction through repetitive motion and visual feedback

The game is lightweight, responsive, and playable on both desktop and mobile devices.
